# Anthropic Setup

Get an API key so brian can use Claude.

## 1. Create an account

- Go to [console.anthropic.com](https://console.anthropic.com)
- Sign up or log in

## 2. Create an API key

- Go to **Settings** → **API Keys**
- Click **Create Key**
- Copy the key (`sk-ant-...`)
- This is your `ANTHROPIC_API_KEY`

## 3. Add credits

Anthropic's API is prepaid. Make sure your account has credits loaded under **Settings** → **Billing**.
